# حسابات ألعاب الورق (Score)

تطبيق ويب عربي بالكامل (RTL) لحساب نقاط ألعاب الورق الشعبية: **الليخة، الهند الفردي، الهند الشراكة، التركس، والكومبلكس** — مع حفظ المباريات وإحصائيات اللاعبين، يعمل بدون إنترنت ويُثبَّت كتطبيق على الجوال (PWA).

## التقنيات

- React 18 + TypeScript + Vite
- TailwindCSS (دعم RTL ودارك مود)
- Zustand مع persist (localStorage)
- React Router (HashRouter)
- vite-plugin-pwa
- lucide-react للأيقونات

## التشغيل

```bash
npm install
npm run dev       # خادم تطوير على http://localhost:5173
npm run build     # بناء للإنتاج في dist/
npm run preview   # معاينة البناء
```

## التثبيت على الجوال (Android)

1. ابنِ التطبيق ونشره على أي استضافة (Netlify, Vercel, GitHub Pages, ...).
2. افتح الرابط في **Chrome** على الأندرويد.
3. من القائمة → "إضافة إلى الشاشة الرئيسية".

## ملخص قواعد الحساب المُنفَّذة

### ليخة (Hearts)

كل جولة:

| البطاقة | النقاط |
| --- | --- |
| كل قلب ♥ | 1 نقطة (المج 13) |
| بنت السباتة Q♠ | 13 نقطة |
| 10 الديناري 10♦ | 10 نقاط |
| **مجموع الجولة** | **36 نقطة** |

اللعبة بتنتهي لمّا أي لاعب يوصل لـ **101 نقطة**، والفائز هو **الأقل نقاطاً**.

ميزة "الصعود للقمر" 🌙: إذا لاعب أخذ كل القلوب الـ13 + بنت السباتة، بياخد 0 وكل الباقيين بياخدوا 36.

### هند فردي (1×1) / هند شراكة (2×2)

- المباراة 9 جولات، أول من يربح **5 جولات** يفوز بالمباراة.
- الأقل نقاطاً يفوز عند انتهاء المباراة.
- في كل جولة، الفائز يختار نوعها:

| النوع | الفائز | الخاسر |
| --- | --- | --- |
| عادية | −20 | + ورق ×1 |
| هند 100 | −40 | + ورق ×1 |
| هند 150 | −40 | + ورق ×1.5 |
| هند 200 | −40 | + ورق ×2 |

"نقاط الورق" = مجموع نقاط البطاقات التي أخذها الفريق الخاسر بهذه الجولة.

### تركس — 5 عقود × 4 جولات = 20 جولة

| العقد | النوع |
| --- | --- |
| الملك | -75 لمن يأخذه |
| البنات | -25 لكل بنت |
| الدياميند | -10 لكل ديناري |
| اللطوش | -15 لكل يدة |
| تركس | +200/+150/+100/+50 حسب ترتيب الإنهاء |

التتويج: مضاعفة ×2.

### كومبلكس

مرحلة ليخة كاملة (20 جولة) ثم مرحلة تركس كاملة (20 جولة) = 40 جولة، والمجموع التراكمي يحدد الفائز.

## الميزات

- واجهة عربية بالكامل + RTL.
- وضع داكن/فاتح.
- إدخال موجه مع تحقق من صحة المجاميع.
- تراجع آخر جولة.
- تاريخ المباريات وإحصائيات اللاعبين.
- يعمل بدون إنترنت بعد أول تحميل.
- يحفظ كل البيانات محلياً (localStorage).
